APSIM (next generation)

Version 2015.06
 June 2015

  • Added a 'replacements' folder option where models dropped on this will replace models of the same name and type in the .apsimx file
  • Numerous improvements to the graphing capabilities of the user interface.
  • Numerous changes to the Plant Modelling Framework (PMF) in readiness for new crop models.
  • Improvements to the auto-documentation functionality.
  • Summary component now works on LINUX and OSX
  • Fixed a factorial defect relating to manager scripts.
  • Fixed user interface move up/down bug.
  • Fixed defect where the weather component in the user interface wasn't updating after changing the weather file.

Version 2014.11
 November 2014

  • Fixed problems with editing information in soil grids in user interface.
  • Fixed reporting of warnings and errors to summary file.
  • Fixed a range of graph editing issues including slow down of graph redraws.
  • Improved performance caused by unnecessary compiling of manager scripts.
  • A first iteration of a sugar cane model - not for use yet.
  • Fixed several LINUX and OSX issues when running under MONO.
  • Added an improved OilPalm example.

Version 2014.10
 October 2014

This is the first release of a new generation of APSIM. It has a limited set of functionality and is intended as a preview. For more information visit the APSIM (next generation) web site
